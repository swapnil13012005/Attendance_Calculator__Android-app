# Attendance Calculator - Android App

A simple and intuitive **Attendance Percentage Calculator** for college students.  
It helps you track your attendance, calculate the number of classes you can bunk,  
or determine how many classes you need to attend to maintain a target percentage.

---

## Features

- Calculate your **current attendance percentage**.
- Determine **how many classes you can safely bunk**.
- Calculate **how many more classes you need to attend** to reach your target percentage.
- Supports **multiple target percentages**: 60%, 65%, 70%, 75%, 80%, 85%, 90%.
- Clean and modern **UI with light and dark theme support**.
- Input validation to ensure **proper values only**.
